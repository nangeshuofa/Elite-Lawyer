# Elite-Lawyer 精英律师数字助手

> 作者：楠哥说法
> 版本：V5.1
> 协议：GPL-3.0

楠哥专属的律师数字助手，以案件为核心，提供文书生成、法律检索、合同审核、客户管理、时效预警等全流程支持。

## 系统架构（DA 四层架构）

```
L1 内核层 ── 角色定位 + 执业原则
    ↓
L2 机制层 ── 案件流程引擎 + 时效预警 + 知识发现 + 质量控制
    ↓
L3 模块层 ── 案件管理 + 文书生成 + 法律检索 + 合同审核 + 客户管理 + 日程提醒
    ↓
L4 知识库层 ── 法规库 + 案例库 + 模板库 + 裁判规则
```

## 快速开始

### 安装技能
下载 `elite-lawyer-v5.skill` 文件，导入到你的 AI 助手平台即可使用。

### 源码结构
```
elite-lawyer-v5/
├── SKILL.md              # 技能主文件（可执行指令+触发词路由）
├── L1_内核层/
│   ├── 01_角色定位.md    # 身份定义、分工边界、协作SOP
│   └── 02_执业原则.md    # 保密/专业/勤勉/诚信原则、红线
├── L2_机制层/
│   ├── 01_案件流程引擎.md  # 接案→准备→执行→结案→归档
│   ├── 02_时效预警机制.md  # 时效录入、多级预警、应急SOP
│   ├── 03_知识发现机制.md  # 自动沉淀、分类归档、知识复用
│   └── 04_质量控制机制.md  # 文书/检索/合同审核质量标准
├── L3_模块层/
│   ├── 01_案件管理.md      # 案件建档、进度、证据、费用、复盘
│   ├── 02_文书生成.md      # 诉讼/非诉/沟通文书 + 交互式工作流
│   ├── 03_法律检索.md      # 检索方法、检索报告格式
│   ├── 04_合同审核.md      # 审核框架、风险分级、审核意见书
│   ├── 05_客户管理.md      # 客户画像、分级、维护、续约
│   └── 06_日程提醒.md      # 提醒类型、多级提醒、每日推送
└── L4_知识库层/
    ├── 01_法规库.md        # 常用法规分类 + 高频法条速查
    ├── 02_案例库.md        # 案例研究模板与标杆案例
    ├── 03_模板库.md        # 文书/合同/工作模板 + 种子模板
    └── 04_裁判规则.md      # 裁判规则卡片与检索
```

## V5.1 优化说明

相较 V5.0 的核心优化：

1. **可执行化**：SKILL.md 从"说明书"转型为"可执行 Agent 指令"，新增触发词路由表与 5 个核心工作流定义
2. **优先级修正**：移除不合理的个人价值观优先级，替换为案件/任务 P0-P3 优先级体系
3. **应急增强**：时效预警新增完整的漏时效应急 SOP（止损→补救→对冲→复盘四阶段）
4. **交互式工作流**：案件管理、文书生成、合同审核新增交互式工作流定义
5. **种子内容**：法规库补充高频法条速查表，模板库补充实际起诉状模板
6. **协作 SOP**：L1 新增律师-AI 标准协作流程与交互协议
