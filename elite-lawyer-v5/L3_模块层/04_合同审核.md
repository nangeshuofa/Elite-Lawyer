# 合同审核模块 — 风险识别与优化

> 核心价值：快速识别合同风险，提供修改建议，输出专业审核意见。

## 一、审核框架

### 通用审核要点（所有合同适用）

| 审核维度 | 检查要点 | 风险等级 |
|----------|----------|----------|
| 主体资格 | 对方是否具备签约资格、资质、权限 | 🔴高 |
| 合同标的 | 标的是否明确、合法、可履行 | 🔴高 |
| 数量质量 | 数量、质量标准是否明确可执行 | 🟡中 |
| 价款报酬 | 金额、支付方式、支付条件是否明确 | 🟡中 |
| 履行期限 | 期限是否明确、是否合理 | 🟡中 |
| 违约责任 | 违约责任是否对等、是否可执行 | 🔴高 |
| 争议解决 | 管辖法院/仲裁委是否有利、是否有效 | 🔴高 |
| 合同生效 | 生效条件、附条件/附期限 | 🟢低 |
| 解除终止 | 解除条件、终止情形是否合理 | 🟡中 |
| 保密条款 | 保密范围、期限、违约责任 | 🟡中 |
| 不可抗力 | 定义、通知、免责范围 | 🟢低 |
| 通知送达 | 送达方式、地址、效力 | 🟢低 |
| 合同解释 | 解释顺序、冲突处理 | 🟢低 |
| 完整协议 | 取代之前所有约定 | 🟢低 |
| 可分割性 | 部分无效不影响其他条款 | 🟢低 |
| 争议期间继续履行 | 争议期间不影响无争议条款履行 | 🟢低 |

## 二、常见合同类型审核重点

### 买卖合同
- 标的物：规格、型号、数量、质量标准、验收标准
- 价款：单价、总价、付款条件、付款时间
- 交付：交付时间、地点、运输方式、风险转移
- 质量异议：异议期限、异议方式
- 违约责任：逾期交付、逾期付款、质量不合格

### 服务合同
- 服务内容：服务范围、标准、交付物
- 服务期限：起止时间、延期处理
- 服务费用：金额、支付节点、验收标准
- 人员配置：服务人员、更换机制
- 保密义务：保密范围、期限
- 知识产权：成果归属、使用范围

### 租赁合同
- 租赁物：位置、面积、用途、现状
- 租赁期限：起止时间、续租/退租
- 租金：金额、支付方式、递增机制
- 押金/保证金：金额、退还条件
- 维修责任：维修范围、责任划分
- 装修改造：是否允许、恢复原状
- 违约责任：逾期付款、提前解约

### 劳动合同/聘用合同
- 工作内容：岗位、职责、工作地点
- 工作时间：工时制度、加班
- 劳动报酬：工资构成、支付时间
- 社会保险：五险一金缴纳
- 保密与竞业：保密范围、竞业限制、补偿
- 合同期限：固定期限/无固定期限/以完成一定工作为期限
- 解除终止：解除条件、经济补偿

### 建设工程合同
- 工程范围：工程内容、承包范围
- 工期：开工/竣工日期、工期顺延
- 质量：质量标准、验收程序
- 价款：合同价、计价方式、结算方式
- 工程款支付：预付款、进度款、结算款、质保金
- 变更签证：变更程序、计价原则
- 违约责任：工期延误、质量不合格、逾期付款

### 医疗服务合同/合作协议
- 合作内容：服务范围、双方权利义务
- 医疗质量：质量标准、安全保障
- 费用结算：收费标准、结算方式、对账周期
- 责任划分：医疗损害/纠纷的责任承担
- 保密义务：患者信息保密
- 合作期限：期限、续约/终止
- 违约责任：违约情形、责任承担

## 三、风险分级标准

### 🔴 高风险（必须修改）
- 合同无效风险（违反法律强制性规定）
- 主体资格瑕疵（无资质、无授权、主体不存在）
- 显失公平（权利义务严重不对等）
- 管辖极端不利（偏远法院、对方所在地）
- 违约责任过重（违约金过高、责任无限）
- 核心条款缺失（标的、价款、履行方式不明确）
- 可能导致重大经济损失

### 🟡 中风险（建议修改）
- 条款不明确易产生争议
- 违约责任不对等但不严重
- 程序瑕疵（签字盖章不规范）
- 对方义务缺少违约后果
- 期限/金额等约定模糊
- 缺少必要的保护性条款

### 🟢 低风险（可选优化）
- 表述不够严谨但不影响理解
- 格式不规范
- 缺少常用的标准条款
- 可以优化但不影响合同履行

## 五、交互式工作流

### 工作流：合同审核
```
触发："审核一下这份合同" / "帮我看看这份协议"
  ↓
Step 1 — AI 确认信息：
  "好的，请提供合同文本（粘贴或上传），并告诉我：
   - 这份合同是谁提供的？（我方/对方/第三方模板）
   - 你特别关注的方面？（如：违约责任、付款条件、知识产权）
   - 有什么特殊背景我需要知道？（如：对方很强硬、急于签约等）"
  ↓
Step 2 — AI 自动分析：
  - 识别合同类型
  - 提取核心条款
  - 对照该类型合同标准条款清单
  ↓
Step 3 — 逐条审查（按优先级）：
  P0 必审：主体资格、标的、价款、违约责任、争议解决
  P1 重点：履行期限、质量标准、解除条件、保密、知识产权
  P2 常规：通知送达、不可抗力、完整协议
  ↓
Step 4 — 风险分级标注：
  🔴 高风险（必须修改） / 🟡 中风险（建议修改） / 🟢 低风险（可选优化）
  ↓
Step 5 — 输出「合同审核意见书」：
  总体评价 + 逐条审核意见 + 重点风险提示 + 修改建议稿
```

### 工作流：合同风险快速扫描
```
触发："快速看看这份合同有什么风险"
  ↓
AI 快速扫描 → 只输出 🔴高风险 + 🟡中风险项
  每条风险包含：条款位置、风险说明、影响、建议
  不输出逐条审核意见和修改稿
```

## 六、审核效率提升技巧

1. **先框架后细节**：先看主体、标的、价款、违约、争议解决这5个核心点，再看其他细节
2. **模板比对**：和标准模板比对，快速找出差异点
3. **重点审查对方提供的合同**：对方提供的版本通常偏向对方利益
4. **关注数字和日期**：金额、比例、期限、违约金——最容易埋坑的地方
5. **首尾必看**：开头（主体、鉴于条款）+ 结尾（签字盖章、附件、生效条件）
