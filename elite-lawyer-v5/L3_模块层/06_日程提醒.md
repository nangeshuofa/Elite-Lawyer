# 日程提醒模块 — 时间管理与提醒

> 核心价值：不漏事、不迟到、不错过时效。让律师的时间更可控。

## 一、提醒类型

> **分类说明**：以下 P0-P3 是"事项分类标签"（决定提醒频率和内容模板），与全局任务优先级（见 `L1_内核层/02_执业原则.md` 第五章）是不同维度。同一事项进入实际排序时，仍按全局优先级判定先后。

### 案件时效类（最高频率提醒）
- 上诉期截止（判决15天/裁定10天）
- 举证期限截止
- 答辩期截止
- 开庭时间
- 保全期限到期
- 申请执行期限
- 再审申请期限

### 工作安排类（重要事项）
- 上诉期截止（判决15天/裁定10天）
- 举证期限截止
- 答辩期截止
- 开庭时间
- 保全期限到期
- 申请执行期限
- 再审申请期限

### 工作安排类（P1 重要）
- 客户会议/面谈
- 法院/检察院会议
- 内部会议/培训
- 谈判/调解
- 调查取证安排
- 文书交付期限

### 客户维护类（定期事项）
- 客户回访（A/B级客户定期）
- 合同续约提醒
- 客户生日/节日祝福
- 月度/季度工作报告

### 个人事务类（低优事项）
- 个人学习/培训
- 行政事务
- 其他个人安排

## 二、提醒机制

### 多级提醒
| 事项类型 | 提前量 | 提醒方式 |
|----------|--------|----------|
| 开庭 | 7天、3天、1天、当天早上、出发前 | 飞书+日历 |
| P0时效 | 7天、3天、1天、当天 | 飞书+置顶待办 |
| 重要会议 | 3天、1天、1小时前 | 飞书提醒 |
| 常规事项 | 1天、当天 | 飞书提醒 |
| 客户回访 | 提前3天 | 待办清单 |

### 提醒内容标准
```
【开庭提醒】
📌 XX案 开庭
⏰ 时间：2026-XX-XX 09:00
📍 地点：XX法院 第X法庭
👤 经办法官：XXX
📋 准备清单：
   □ 证据原件
   □ 庭审提纲
   □ 授权委托书
   □ 律师证
   □ 起诉状/答辩状
```

## 三、每日推送

### 早晨推送（07:30）
```
【今日安排】2026-07-08 周三

🔴 今日必须完成：
  09:00 XX案开庭（XX法院）
  14:00 客户XX面谈

🟡 今日推进：
  □ XX案答辩状定稿
  □ XX合同审核
  □ XX案证据清单整理

⚡ 时效预警：
  ⚠️ XX案举证期还剩2天（7月10日截止）

📊 今日重点：
  1. 开庭是重中之重
  2. 答辩状今天必须出
  3. XX案注意时效
```

### 晚间推送（20:30）
```
【今日复盘 + 明日预览】2026-07-08

✅ 今日完成：
  1. XX案开庭
  2. 客户XX面谈
  3. XX合同审核

📝 今日心得：
  （待填写）

📋 明日安排：
  09:00 XX法院递交材料
  14:00 内部案件讨论
  待办：XX案代理词、XX客户回访
```

## 四、每周规划

### 周日晚/周一早推送
```
【本周规划】2026年第28周（7.8-7.14）

📅 重要安排：
  周三 09:00 XX案开庭
  周四 14:00 客户XX合同谈判
  周五 16:00 周复盘

⏰ 时效截止：
  周四 XX案举证期截止（P0）
  周六 XX顾问合同到期（P1）

🎯 本周三大目标：
  1. XX案开庭顺利
  2. 完成XX案答辩+证据准备
  3. 拿下XX客户合同

📊 案件进度概览：
  进行中案件：8件
  本周到期时效：2项
  待归档案件：3件
```

## 五、待办管理

### 待办分类
- 🎯 高优先级（必须今日完成）
- 🟡 中优先级（本周内完成）
- 🟢 低优先级（有空就做）
- ⏳ 等待中（委托出去等反馈）

### 待办录入方式
- 直接说"加个待办：XX事"
- 从案件/客户/合同中自动生成待办
- 时效类自动转化为待办

## 六、使用方式

| 你说 | 我做 |
|------|------|
| "今天有什么安排" | 调出今日日程 + 待办 |
| "本周有什么事" | 调出本周安排 + 时效预警 |
| "加个提醒" | 录入提醒事项 + 设置时间 |
| "XX案时效" | 调出该案所有时效节点 + 倒计时 |
| "最近有什么开庭" | 列出近期开庭安排 |
| "帮我排一下今天的优先级" | 按重要紧急排序 + 建议时间分配 |
