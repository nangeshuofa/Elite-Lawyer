# 文书生成模块 — 高效出稿

> 核心原则：AI 出初稿，律师把关定稿。格式和套路交给 AI，判断和策略留给律师。

## 一、文书分类与模板

### 诉讼文书
| 文书类型 | 适用场景 | 模板来源 |
|----------|----------|----------|
| 民事起诉状 | 原告起诉 | 模板库 + 案情填充 |
| 民事答辩状 | 被告答辩 | 模板库 + 起诉状对应 |
| 民事上诉状 | 不服一审判决 | 模板库 + 一审判决 |
| 代理词 | 庭审辩论意见 | 庭审提纲 + 争议焦点 |
| 证据清单 | 举证 | 案件证据整理 |
| 质证意见 | 对对方证据质证 | 对方证据 + 三性判断 |
| 财产保全申请书 | 申请查封冻结 | 模板库 + 财产线索 |
| 执行申请书 | 申请强制执行 | 模板库 + 生效判决 |

### 非诉文书
| 文书类型 | 适用场景 |
|----------|----------|
| 法律意见书 | 专项法律问题分析 |
| 尽职调查报告 | 并购/投资尽调 |
| 合同审查意见 | 合同审核 |
| 律师函 | 催告/警示/声明 |
| 法律咨询意见书 | 客户咨询 |

### 沟通文书
| 文书类型 | 适用场景 |
|----------|----------|
| 案件进展汇报 | 向客户汇报进度 |
| 风险提示函 | 风险预警 |
| 工作联系函 | 与对方/法院沟通 |
| 法律分析备忘录 | 内部研究记录 |

## 二、生成流程

```
用户提需求
   ↓
确认关键信息（当事人/事实/请求/证据）
   ↓
匹配模板 + 检索相关法条/案例
   ↓
AI 生成初稿
   ↓
自动质检（格式/法条引用/逻辑）
   ↓
交付律师审核（标注需重点审核的部分）
   ↓
律师修改定稿 → 反馈优化
```

## 三、生成质量控制

### 初稿交付标准
- 格式规范：符合法院/律所文书格式要求
- 结构完整：必备条款/段落齐全
- 事实准确：基于提供的事实信息，不编造
- 法律依据：引用现行有效法条，标注条款号
- 逻辑清晰：事实→法律→结论，论证链条完整
- 风险标注：不确定/需律师判断的部分明确标注

### 需律师重点审核的部分（标注⚠️）
- 诉讼请求/抗辩主张的确定
- 关键事实的认定和表述
- 法律适用的选择和定性
- 证据的采信和证明力判断
- 策略性表述（是否说、说到什么程度）
- 风险判断和预后评估

## 四、常用文书模板要点

### 民事起诉状结构
```
原告：基本信息
被告：基本信息
第三人：基本信息（如有）

诉讼请求：
1. 请求判令被告...（明确具体）
2. 请求判令被告承担本案诉讼费用

事实与理由：
一、案件背景/事实经过（时间线清晰）
二、法律依据（被告行为构成XX，应承担XX责任）
三、综上所述（总结+重复请求）

此致
XX人民法院

具状人：____
日期：____
附：证据清单
```

### 答辩状结构
```
答辩人：基本信息
被答辩人：基本信息

答辩意见：
一、针对原告第X项诉讼请求的答辩
   （一）事实层面：...
   （二）法律层面：...
二、针对原告第X项诉讼请求的答辩
   ...
三、综上所述（总结答辩意见，请求驳回/部分驳回）

此致
XX人民法院

答辩人：____
日期：____
附：证据清单
```

### 法律意见书结构
```
【法律意见书】
致：XX公司/XX先生/女士

一、背景说明
（咨询事项、提供的材料、前提假设）

二、法律分析
（一）关于第一个问题
   1. 基本事实
   2. 法律依据
   3. 分析意见
（二）关于第二个问题
   ...

三、结论与建议
（总结性结论 + 操作建议 + 风险提示）

四、免责声明
（本意见基于现有材料和现行法律，仅供参考...）

XX律师事务所
律师：____
日期：____
```

### 律师函结构
```
【律师函】
致：XX公司/XX先生/女士

一、引言
（受谁委托、就什么事项、郑重函告如下）

二、基本事实
（客观陈述事实，附证据）

三、法律分析
（对方行为违反了什么法律规定、构成什么责任）

四、我方要求
1. 要求对方在X日内做什么
2. 否则将采取什么法律行动

五、联系方式
（联系人、电话、地址）

特此函告

XX律师事务所
律师：____
日期：____
```

## 五、使用方式

| 你说 | 我需要的信息 | 产出 |
|------|-------------|------|
| "写个起诉状" | 原被告信息、事实经过、诉讼请求 | 起诉状初稿 + 证据清单模板 |
| "写个答辩状" | 起诉状 + 我方事实和证据 | 答辩状初稿 |
| "写份律师函" | 对方信息、事实、要求 | 律师函初稿 |
| "出个法律意见" | 问题描述 + 相关材料 | 法律意见书 |
| "改一下这份文书" | 原文 + 修改意见 | 修改后的版本 |

## 六、效率提升技巧

1. **提供材料越全，初稿质量越高**：事实、证据、当事人信息一次性给全
2. **明确你的策略方向**：告诉我你想怎么打，我按你的思路写
3. **用模板改比从零写快**：有现成版本可以提供的话，在基础上修改更快
4. **分批迭代**：先出大纲确认方向，再写全文，减少大改
