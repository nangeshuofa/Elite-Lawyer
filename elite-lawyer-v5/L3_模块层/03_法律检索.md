# 法律检索模块 — 快速定位依据

> 核心价值：把律师从法条堆里解放出来，快速找到最相关的法律依据和裁判规则。

## 一、检索范围

### 1. 法律法规
- 法律（全国人大及其常委会制定）
- 行政法规（国务院制定）
- 司法解释（最高法、最高检）
- 部门规章（国务院各部门）
- 地方性法规/地方政府规章
- 行业规范/技术标准

### 2. 司法案例
- 最高院指导案例
- 最高院典型案例
- 各省高院参考案例
- 同类案件裁判文书

### 3. 裁判规则
- 最高院司法观点
- 审判纪要/指导意见
- 类案裁判要旨
- 法官会议纪要

### 4. 学术观点
- 权威学者论著
- 主流学理观点
- 争议焦点梳理

## 二、检索方法

### 关键词检索
- 直接用法律概念/争议点做关键词
- 同义词/近义词扩展
- 法条原文关键词

### 法条关联检索
- 找到核心法条 → 查看相关司法解释 → 查看适用案例
- 上下位法关联、特别法与一般法关联

### 案例反向检索
- 找到典型案例 → 看适用了哪些法条 → 扩展法条体系

### 分类检索
- 按法律部门：民法/刑法/行政法/劳动法...
- 按案由：合同纠纷/侵权纠纷/劳动争议...
- 按法院层级：最高院/高院/中院...

## 三、检索报告标准格式

## 四、常见领域检索要点

### 医疗损害责任
- 核心法条：《民法典》侵权责任编 第六章
- 司法解释：《医疗损害责任纠纷司法解释》
- 关键问题：过错认定、因果关系、举证责任、鉴定标准
- 检索重点：司法鉴定规则、过错参与度、赔偿标准

### 劳动争议
- 核心法条：《劳动合同法》《劳动争议调解仲裁法》
- 关键问题：劳动关系认定、劳动合同效力、工伤认定、经济补偿金
- 检索重点：地方裁判口径、指导意见

### 合同纠纷
- 核心法条：《民法典》合同编
- 关键问题：合同效力、违约认定、损失计算、解除条件
- 检索重点：最高院指导案例、典型判例

## 五、使用方式

| 你说 | 我做 |
|------|------|
| "查一下XX法律问题" | 检索法条+案例 → 出检索报告 |
| "找一下XX法第X条" | 定位法条原文 + 相关司法解释 |
| "有没有类似案例" | 类案检索 + 裁判规则提炼 |
| "这个问题主流观点是什么" | 学理观点 + 裁判倾向汇总 |
| "XX案由的裁判标准" | 裁判规则 + 典型案例 |

## 六、检索质量原则

1. **来源可查**：所有引用都标注来源（法条全称+条款号/案号+法院）
2. **现行有效**：确认引用的法律法规现行有效，未被废止
3. **区分层级**：法律 > 司法解释 > 指导案例 > 普通案例 > 学理观点
4. **客观中立**：有利和不利的都列，不挑好听的说
5. **说明局限**：明确说明检索范围和可能的遗漏
