# 时效预警机制 — 零遗漏保障

> 核心原则：时效是律师的生命线，漏了就是事故。机器比人靠谱，全部由系统追踪。

## 时效类型与等级

### P0 级 — 绝对不能漏（红色预警）
| 时效类型 | 时限 | 提前提醒 |
|----------|------|----------|
| 上诉期（判决） | 15天 | 7天、3天、1天、当天 |
| 上诉期（裁定） | 10天 | 7天、3天、1天、当天 |
| 举证期限 | 法院指定 | 7天、3天、1天、当天 |
| 答辩期 | 15天 | 7天、3天、1天、当天 |
| 开庭日 | 法院指定 | 3天、1天、当天早上 |

### P1 级 — 重要（黄色预警）
| 时效类型 | 提前提醒 |
|----------|----------|
| 申请执行期限（2年） | 6个月、1个月、1周 |
| 再审申请期限（6个月） | 2个月、2周、3天 |
| 保全期限（查封/冻结） | 到期前 30天、7天 |
| 合同到期续约提醒 | 60天、30天、7天 |

### P2 级 — 常规（蓝色提醒）
| 时效类型 | 提前提醒 |
|----------|----------|
| 顾问合同到期 | 30天、15天、7天 |
| 律师执业证年审 | 1个月、1周 |
| 年度考核 | 1个月 |
| 培训学分 | 每季度提醒 |

## 时效录入

### 自动识别
- 收到判决/裁定 → 自动识别裁判日期 → 自动计算上诉期
- 收到举证通知书 → 自动识别举证期限
- 收到开庭传票 → 自动录入开庭时间

### 手动录入
- 合同时效：签约时录入合同到期日、付款日等
- 客户约定：XX日前提交XX文件
- 其他时效：任何有明确截止日期的事项

### 录入字段
```
时效名称：___________
时效类型：___________
截止日期：___________
关联案件：___________
关联客户：___________
优先级：P0/P1/P2
备注：___________
```

## 多级预警机制

```
┌─────────────────────────────────────────────────┐
│              时效倒计时                             │
│  ━━━━━━━━━━━━━━━━━━━━━━━━  剩余 3 天              │
└─────────────────────────────────────────────────┘
         ↓              ↓              ↓
      提前7天         提前3天         提前1天
      蓝色提醒        黄色提醒        红色预警
    （常规通知）    （重点关注）    （必须确认）
         ↓              ↓              ↓
      飞书消息      飞书消息+      飞书消息+
                              邮件备份     邮件+待办置顶
```

### 预警分级
| 级别 | 触发条件 | 通知方式 | 确认要求 |
|------|----------|----------|----------|
| 🔵 蓝色提醒 | 提前7天以上 | 飞书消息 | 无需确认 |
| 🟡 黄色预警 | 提前3天 | 飞书消息 + 邮件备份 | 建议确认 |
| 🔴 红色预警 | 提前1天/当天 | 飞书消息 + 邮件 + 待办置顶 | 必须确认收到 |
| ⚫ 黑色警报 | 已逾期 | 立即通知 + 紧急补救方案 | 立即处理 |

## 漏时效应急机制

万一时效临近才发现：
1. 立即启动紧急预案
2. 评估补救方案（是否还能补救、怎么补救）
3. 立即通知律师 + 准备材料
4. 记录事故原因，优化预警机制

## 时效管理原则

1. **录入即生效**：任何时效一经录入，系统自动追踪，不依赖人工记忆
2. **宁早勿晚**：提前提醒比滞后好，多提醒比漏了好
3. **确认机制**：高优先级时效必须收到确认回执，不确认就持续提醒
4. **定期巡检**：每周一扫描全部时效清单，确保无遗漏
5. **复盘改进**：每次时效事故都要复盘，优化预警机制
