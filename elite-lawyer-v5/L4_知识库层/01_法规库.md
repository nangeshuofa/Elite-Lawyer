# 法规库 — 常用法律法规汇编

> 知识库的基础弹药库，按领域分类，随办案持续积累更新。

## 一、法规分类体系

### 宪法及宪法相关法
- 中华人民共和国宪法
- 立法法、选举法等

### 民商法
- 民法典（总则、物权、合同、人格权、婚姻家庭、继承、侵权责任）
- 公司法、合伙企业法、个人独资企业法
- 证券法、票据法、保险法、海商法
- 著作权法、专利法、商标法

### 行政法
- 行政许可法、行政处罚法、行政强制法
- 行政复议法、行政诉讼法
- 政府信息公开条例
- 各行业行政法规（医疗卫生、环境保护等）

### 经济法
- 反垄断法、反不正当竞争法
- 产品质量法、消费者权益保护法
- 劳动法、劳动合同法、社会保险法
- 税法（企业所得税法、个人所得税法等）

### 社会法
- 社会保险法
- 社会救助法
- 公益事业捐赠法

### 刑法
- 中华人民共和国刑法
- 刑法修正案（一）至（十二）

### 诉讼与非诉讼程序法
- 民事诉讼法
- 刑事诉讼法
- 行政诉讼法
- 仲裁法
- 人民调解法

## 二、重点领域常用法规

### 医疗法律
**核心法律：**
- 《中华人民共和国基本医疗卫生与健康促进法》
- 《中华人民共和国医师法》
- 《中华人民共和国药品管理法》
- 《中华人民共和国传染病防治法》

**核心法规：**
- 《医疗纠纷预防和处理条例》
- 《医疗机构管理条例》
- 《医疗事故处理条例》
- 《护士条例》

**核心规章/规范：**
- 《病历书写基本规范》
- 《医疗机构病历管理规定》
- 《医疗技术临床应用管理办法》
- 《人体器官移植条例》

**民法典相关：**
- 《民法典》第七编 侵权责任 第六章 医疗损害责任
- 第一千二百一十八条 ~ 第一千二百二十八条

### 劳动法
**核心法律：**
- 《中华人民共和国劳动法》
- 《中华人民共和国劳动合同法》
- 《中华人民共和国劳动争议调解仲裁法》
- 《中华人民共和国社会保险法》
- 《工伤保险条例》

**核心司法解释：**
- 最高人民法院关于审理劳动争议案件适用法律问题的解释（一）
- 最高人民法院关于审理工伤保险行政案件若干问题的规定

### 合同法
**核心法律：**
- 《民法典》第三编 合同
  - 第一分编 通则
  - 第二分编 典型合同（19类有名合同）

**核心司法解释：**
- 最高人民法院关于适用《中华人民共和国民法典》合同编通则若干问题的解释
- 买卖合同司法解释、民间借贷司法解释等

### 侵权责任法
**核心法律：**
- 《民法典》第七编 侵权责任
  - 一般规定
  - 损害赔偿
  - 责任主体的特殊规定
  - 产品责任
  - 机动车交通事故责任
  - 医疗损害责任
  - 环境污染和生态破坏责任
  - 高度危险责任
  - 饲养动物损害责任
  - 建筑物和物件损害责任

## 五、法规更新机制

### 动态更新
- 新法发布 → 及时收录并标注
- 法规修订 → 同步更新，标注修改点
- 法规废止 → 标记废止，保留历史版本备查

### 重点关注
- 最高法司法解释（办案最常用）
- 行政法规（行业监管重要依据）
- 地方性法规（本地区特别规定）

## 六、检索方法

1. **按领域查找**：先确定法律部门 → 具体领域 → 核心法条
2. **按关键词检索**：直接搜索关键词 → 定位相关法条
3. **按案例反推**：从案例引用的法条 → 扩展相关法条体系
4. **上下关联**：找到核心法条 → 查看配套的司法解释、实施细则
