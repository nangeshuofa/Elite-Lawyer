# 裁判规则库 — 分类说明与导航

> 本文件为裁判规则库的分类说明。**实际规则内容已按领域分片至 `裁判规则/` 子目录**，AI 检索时请先读 `index.md` 定位分片文件。

## 规则分类体系（索引）

### 按领域 → `裁判规则/` 子目录
- 医疗损害 → `裁判规则/医疗损害.md`（已含：过错认定/因果关系/举证/知情同意）
- 劳动争议 → `裁判规则/劳动争议.md`（已含：劳动合同/工伤/经济补偿）
- 合同纠纷 → `裁判规则/合同纠纷.md`（已含：效力/违约/定金）

## 裁判规则卡片格式
```markdown
### [类型] 规则名称
- **来源**：法条/案号
- **领域**：医疗/劳动/合同/...
- **摘要**：一句话核心结论
- **详情**：具体内容（可引原文）
- **法律依据**：相关法条/司法解释
- **复用场景**：何时可再次使用
- **添加日期**：YYYY-MM-DD
```

## 规则更新机制
1. 新法/新司法解释发布 → 补充新增规则，废止/修改旧规则
2. 新案例出现 → 提炼新规则，补充案例支撑
3. 裁判观点变化 → 标注变化，说明新旧差异
4. 定期整理 → 每季度去重、合并、更新

## 使用方式
| 你说 | 我做 |
|------|------|
| "医疗过错怎么认定" | 调 `裁判规则/医疗损害.md` + 案例 |
| "XX问题法院怎么判" | 检索相关分片 + 典型案例 |
| "最高院关于XX的指导意见" | 调出原文 + 适用要点 |
