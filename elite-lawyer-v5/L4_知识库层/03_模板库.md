## 四、种子模板：民事起诉状（通用版）

> 以下为 AI 生成起诉状时的基础模板。`【】` 内为需填充的内容。

```
民事起诉状

原告：【姓名/名称】，【性别】，【出生日期/统一社会信用代码】，
住址/住所地：【地址】，联系方式：【电话】。

被告：【姓名/名称】，【性别】，【出生日期/统一社会信用代码】，
住址/住所地：【地址】，联系方式：【电话】。

（如有第三人，格式同上）

诉讼请求：
1. 请求判令被告【具体请求，如：赔偿原告医疗费、误工费、护理费等各项损失共计人民币XX元】；
2. 请求判令被告【第二项请求，如：承担本案诉讼费用】。

事实与理由：

一、案件基本事实
【时间线清晰陈述事实经过。包括但不限于：
- 当事人之间的关系
- 关键事件的发生时间、地点、经过
- 造成的后果/损失
- 已采取的措施（协商、调解等）】

二、法律依据
【被告的行为/事实】违反了《【法律名称】》第【X】条之规定：【引用法条关键内容】。
根据《【法律名称】》第【X】条，被告应承担【责任类型】。

三、综上所述
【总结 + 重复诉讼请求】

此致
【XX】人民法院

具状人（签名/盖章）：_________
日期：____年____月____日

附：
1. 本诉状副本____份
2. 证据清单及证据材料____份
```

## 五、模板使用原则

1. **模板是起点，不是终点**：每个案件都不同，模板要根据具体情况调整
2. **核心内容不能套**：事实认定、法律适用、策略选择——这些必须律师自己判断
3. **格式可以套**：文书格式、标准条款、程序性内容——直接用模板提高效率
4. **持续优化模板**：每次使用后，如果发现模板可以改进，就更新模板

## 六、模板贡献机制

每次写完一份满意的文书：
1. 判断是否值得做成模板（可复用性）
2. 提取通用结构 + 填空位置
3. 添加使用说明和注意事项
4. 归档到模板库对应分类
5. 更新模板索引

→ 好模板省下来的时间，下次办案直接用。

## 七、使用方式

| 你说 | 我做 |
|------|------|
| "给我一个起诉状模板" | 调出通用起诉状模板 |
| "医疗损害的起诉状怎么写" | 调出专用模板 + 写作要点 |
| "合同审核意见书模板" | 调出模板 + 填写指引 |
| "更新XX模板" | 根据反馈修改模板 |
| "这个文书做得好，做成模板" | 提取模板 → 归档 → 更新索引 |
