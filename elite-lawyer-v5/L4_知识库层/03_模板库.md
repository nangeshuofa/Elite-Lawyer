# 模板库 — 分类说明与导航

> 本文件为模板库的分类说明。**实际模板内容已分片至 `模板库/` 子目录**，AI 生成文书时请先读 `index.md` 定位分片文件。

## 模板分类体系（索引）

### 诉讼文书 → `模板库/诉讼文书.md`
- 起诉状（通用/专用）、答辩状、上诉状、代理词
- 证据清单、质证意见、财产保全申请书、执行申请书

### 合同文书 → `模板库/合同文书.md`
- 买卖/租赁/服务/劳动/合作/股权合同
- 保密协议、竞业限制协议

### 非诉文书（暂存本文件，待分片）
- 法律意见书、尽职调查报告、律师函

### 内部工作模板（暂存本文件，待分片）
- 案件接案评估表、进度周报、结案报告、复盘模板
- 客户访谈记录、满意度调查、年度服务报告
- 每日计划、每周总结、检索报告、合同审核意见书

## 模板使用原则
1. 模板是起点不是终点：每个案件不同，模板需按情况调整
2. 核心内容不能套：事实认定/法律适用/策略必须律师判断
3. 格式可以套：文书格式/标准条款/程序性内容直接用
4. 持续优化：用后若可改进则更新模板

## 模板贡献机制
写完满意文书 → 判断可复用性 → 提取通用结构+填空位 → 添加使用说明 → 归档对应分片 → 更新 `index.md`
