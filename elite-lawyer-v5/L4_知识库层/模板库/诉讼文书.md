# 模板库 · 诉讼文书分片

> 按 `index.md` 检索协议访问。本文件收录诉讼类文书模板。

## 种子模板：民事起诉状（通用版）

> `【】` 内为需填充内容。AI 生成起诉状时基于此模板 + 案情填充。

```
民事起诉状

原告：【姓名/名称】，【性别】，【出生日期/统一社会信用代码】，
住址/住所地：【地址】，联系方式：【电话】。

被告：【姓名/名称】，【性别】，【出生日期/统一社会信用代码】，
住址/住所地：【地址】，联系方式：【电话】。

（如有第三人，格式同上）

诉讼请求：
1. 请求判令被告【具体请求，如：赔偿原告医疗费、误工费、护理费等各项损失共计人民币XX元】；
2. 请求判令被告【第二项请求，如：承担本案诉讼费用】。

事实与理由：

一、案件基本事实
【时间线清晰陈述事实经过。包括：当事人关系、关键事件发生时间地点经过、造成后果/损失、已采取措施】

二、法律依据
【被告行为】违反了《【法律名称】》第【X】条之规定：【引用法条关键内容】。
根据《【法律名称】》第【X】条，被告应承担【责任类型】。

三、综上所述
【总结 + 重复诉讼请求】

此致
【XX】人民法院

具状人（签名/盖章）：_________
日期：____年____月____日

附：
1. 本诉状副本____份
2. 证据清单及证据材料____份
```
