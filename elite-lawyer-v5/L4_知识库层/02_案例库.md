# 案例库 — 分类说明与导航

> 本文件为案例库的分类说明。**实际案例内容已分片至 `案例库/` 子目录**，AI 检索时请先读 `index.md` 定位分片文件。

## 案例分类体系（索引）

### 按领域 → 各分片文件内按标签区分
- 医疗损害纠纷 / 劳动争议 / 合同纠纷 / 公司事务 / 行政 / 执行 / 刑事

### 按结果
- 胜诉案例 / 部分胜诉 / 败诉案例 / 调解案例 / 撤诉案例

### 按价值
- ⭐⭐⭐ 标杆案例 → `案例库/标杆案例.md`
- ⭐⭐ 重要案例
- ⭐ 一般案例
- 败诉案例专题 → `案例库/败诉案例.md`

## 案例研究标准格式
```
【案例研究】案例名称
一、基本信息：案号/法院/法官/案由/裁判日期/我方角色/结果
二、案情摘要（200字内）
三、争议焦点
四、我方策略与证据链
五、对方观点与证据
六、裁判结果
七、经验教训（✅/❌/💡）
八、可复用知识
九、关联信息（标签/法条/路径）
```

## 检索维度
- 按领域 / 法院层级 / 案由 / 争议焦点 / 关键词 / 结果

> 标杆案例和败诉案例的实际内容见 `案例库/` 子目录对应文件。
