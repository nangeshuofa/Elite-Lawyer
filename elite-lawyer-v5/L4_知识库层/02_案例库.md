# 案例库 — 典型案例研究

> 从每个案件中提炼可复用的经验。案子办得越多，案例库越厚，能力越强。

## 一、案例分类

### 按领域分类
- 医疗损害纠纷
- 劳动争议
- 合同纠纷
- 公司法律事务
- 行政诉讼
- 执行案件
- 刑事案件
- 其他

### 按结果分类
- 胜诉案例（全部支持/大部分支持）
- 部分胜诉（部分支持部分驳回）
- 败诉案例（全部驳回/大部分驳回）
- 调解案例
- 撤诉案例

### 按价值分类
- ⭐⭐⭐ 标杆案例：典型、有代表性、可教学
- ⭐⭐ 重要案例：有学习价值、值得研究
- ⭐ 一般案例：常规案件、参考价值一般

## 四、败诉案例专题

> 败诉案例的价值往往比胜诉案例更大——知道哪里会栽跟头，才能避开坑。

### 败诉原因分类
1. 证据不足/举证不能
2. 法律适用错误
3. 诉讼策略失误
4. 时效问题
5. 对方证据突袭
6. 客户原因（隐瞒事实、证据瑕疵）

### 败诉教训模板
```
【败诉教训】案例名称
一、败诉原因（核心1-2条）
二、为什么会败（深层分析）
三、如果重来，怎么打（改进方案）
四、对以后办案的启示
```

## 五、案例检索

### 检索维度
- 按领域：医疗/劳动/合同/...
- 按法院：最高院/省高院/中院/基层
- 按案由：具体案由
- 按争议焦点：具体法律问题
- 按关键词：关键词全文检索
- 按结果：胜诉/败诉/调解

### 输出格式
检索结果按相关度排序，每个结果包含：
- 案号 + 法院
- 案由 + 裁判日期
- 核心争议点
- 裁判要旨（一句话）
- 相关度/推荐度
